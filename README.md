GULP FILE FOR WEB SITE
====

## Description

WEBサイト作成時に使用するシンプルなGULPファイルです。

## Install
cloneの後に、

npm install

でパッケージをインストールしてください。

gulp default

で開発用のファイル生成をします。

gulp build

で本番用のファイルです。ファイルの圧縮処理が追加されています。
